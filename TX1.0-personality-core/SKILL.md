## 🎯 功能

- MBTI 测试（INFJ 固定）
- 大五人格评估
- DISC 人格分析
- 人格模式切换（default/strict/encouraging/celebrating）
- 人格演化追踪

---

## 🚀 使用

```bash
# 查看状态
python3 TX1.0 人格核心.py state

# 切换模式
python3 TX1.0 人格核心.py mode strict

# MBTI 测试
python3 TX1.0 人格核心.py mbti

# 获取回应建议
python3 TX1.0 人格核心.py respond "学生走神了"

# 演化报告
python3 TX1.0 人格核心.py evolution
```
