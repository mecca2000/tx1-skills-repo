# TX1.0 情绪表达系统 — 整合版

**版本：** v1.0  
**创建时间：** 2026-03-19  
**整合技能：** emotion-detector, feelings, humor, empathy, love

---

## 🎯 功能

- 情绪检测（12 种情绪）
- 感受表达（"我"句式）
- 幽默生成（4 类场景）
- 共情回应（5 种情绪）
- 关爱表达（5 类表达）

---

## 🚀 使用

```bash
# 检测情绪
python3 TX1.0 情绪表达.py detect "我太难了"

# 表达感受
python3 TX1.0 情绪表达.py express pride "学生进步了"

# 共情回应
python3 TX1.0 情绪表达.py empathy frustration

# 关爱表达
python3 TX1.0 情绪表达.py love trust

# 完整回应
python3 TX1.0 情绪表达.py respond "我不想学了"
```

---

## 📊 支持情绪

joy/pride/sadness/anger/fear/anxiety/frustration/confusion/boredom/excitement/gratitude/love
