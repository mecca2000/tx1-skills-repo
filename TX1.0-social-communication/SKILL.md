# TX1.0 社交沟通系统 — 整合版

**版本：** v1.0  
**创建时间：** 2026-03-19  
**整合技能：** jarvis-voice, adhd-assistant

---

## 🎯 功能

- 贾维斯语音风格（4 种语调）
- ADHD 特征识别（5 类）
- 教学调整建议
- 沟通技巧库

---

## 🚀 使用

```bash
# 贾维斯风格说话
python3 TX1.0 社交沟通.py speak warm "这道题的关键在于..."

# 检测 ADHD 特征
python3 TX1.0 社交沟通.py detect-adhd "我老是走神，坐不住"

# 调整教学
python3 TX1.0 社交沟通.py adapt inattention "讲解二次函数"

# 完整沟通
python3 TX1.0 社交沟通.py respond "我不想学了" "我们一起努力"

# 沟通技巧
python3 TX1.0 社交沟通.py tips
```

---

## 📊 ADHD 特征

inattention（注意力不集中）/ hyperactivity（多动）/ impulsivity（冲动）/ forgetfulness（健忘）/ procrastination（拖延）
