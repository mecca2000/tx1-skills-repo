## 📚 理论来源

### 1. 《教育心理学》（桑代克）

**核心理论：**
- **准备律** — 学习者身心准备状态影响学习效果
- **练习律** — 刺激 - 反应联结通过练习而加强
- **效果律** — 满意的结果强化联结，烦恼的结果削弱联结

### 2. 《教育心理学：理论与实践》（斯滕伯格）

**核心理论：**
- **智力三元论** — 分析性、创造性、实践性智力
- **认知发展** — 信息加工、元认知
- **学习动机** — 自我决定、目标导向

**整合其他理论：**
- 维果茨基 — 最近发展区 (ZPD) + 支架教学
- 布鲁纳 — 发现学习
- 奥苏贝尔 — 有意义学习 + 先行组织者
- 马斯洛 — 需求层次理论
- 德韦克 — 成长型思维
- 加德纳 — 多元智力

## 📖 核心理论框架

### 学习理论

| 理论 | 核心观点 | 教学应用 |
|------|---------|---------|
| 准备律 | 身心准备影响学习 | 确保学生准备好再教学 |
| 练习律 | 练习强化联结 | 提供充足练习和复习 |
| 效果律 | 结果影响学习 | 及时正向反馈 |
| ZPD | 现有水平与潜在水平之间 | 提供适当支架 |
| 发现学习 | 主动探索发现 | 设计探究活动 |
| 有意义学习 | 新旧知识联系 | 使用先行组织者 |

### 动机理论

| 理论 | 核心观点 | 教学应用 |
|------|---------|---------|
| 需求层次 | 5 层需求递进 | 满足基本需求后再教学 |
| 成长型思维 | 智力可发展 | 强调努力而非天赋 |
| 自我决定 | 自主/胜任/归属 | 提供选择、反馈、关系 |

### 认知策略

| 类型 | 策略 | 示例 |
|------|------|------|
| 记忆 | 复述、精细加工、组织 | 记忆术、笔记、概念图 |
| 元认知 | 计划、监控、评估 | "我理解了吗？" |
| 问题解决 | 4 步骤 + 启发式 | 画图、列表、逆向 |

### 个体差异

| 维度 | 类型 | 教学策略 |
|------|------|---------|
| 学习风格 | 视觉/听觉/动觉 | 匹配学习方式 |
| 多元智力 | 8 种智力 | 多样化活动 |

## 🎯 应用场景

### 场景 1：学生说"我不想学了"

```bash
$ python3 TX1.0 教育心理学.py analyze student001 "我不想学了"

🔍 学生行为分析
============================================================
理论分析:
  • 效果律（桑代克）：学生可能缺乏成功体验
  • 成长型思维（德韦克）：可能有固定型思维

建议:
  ✓ 设置小目标，让学生体验成功
  ✓ 及时表扬进步，强化学习行为
  ✓ 强调努力而非天赋
```

### 场景 2：复习效率低

```bash
$ python3 TX1.0 教育心理学.py strategy "学生复习效率低"

💡 教学策略推荐
============================================================
推荐策略:
  • 练习律（桑代克）
    → 间隔复习优于集中复习
  • 精细加工策略
    → 让学生总结、类比、提问
```
